1. Jogo de Adivinhação
Gere um número aleatório entre 1 e 100. Peça ao usuário para adivinhar o número, dando dicas ("maior" ou "menor") a cada tentativa. Armazene todas as tentativas em uma lista e mostre o histórico ao final.

2. Calculadora de Média com Aprovação
Crie um programa que receba uma lista de notas (via input) até o usuário digitar -1. Calcule a média das notas e informe se o aluno foi aprovado (média ≥ 7), reprovado (média < 5) ou está em recuperação (5 ≤ média < 7).

3. Filtro de Números Pares e Ímpares
Crie uma lista com números digitados pelo usuário até ele digitar 0. Separe os números em duas listas (pares e ímpares) e exiba as duas listas no final.

4. Simulador de Caixa Eletrônico
Crie um programa que simule saques. O usuário informa o valor a sacar, e o programa deve informar as notas (100, 50, 20, 10) necessárias, usando um dicionário para armazenar a quantidade de cada nota disponível. Use loops e condições para validar saques.
