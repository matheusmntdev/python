Laços de Repetição em Python

Os laços de repetição (ou loops) são estruturas que permitem executar um bloco de código várias vezes, seja um número fixo de vezes ou enquanto uma condição específica for verdadeira. Eles são essenciais para automatizar tarefas repetitivas, como processar listas de dados, gerar sequências ou esperar por uma ação do usuário.

Em Python, os dois principais laços de repetição são:

  • for: Ideal para iterar sobre uma sequência conhecida, como uma lista, tupla, string ou range.
  • while: Usado quando o número de iterações não é conhecido antecipadamente e depende de uma condição.

Além disso, existem comandos como break e continue que ajudam a controlar o fluxo dentro dos laços.

1. Laço for

O laço for é usado para iterar sobre uma sequência (como uma lista, tupla, string ou range) ou qualquer objeto iterável. Ele executa o bloco de código uma vez para cada item na sequência.

Sintaxe:

for item in sequência:
    # Bloco de código a ser executado para cada item
  • item: Variável que assume o valor de cada elemento da sequência a cada iteração.
  • sequência: Uma coleção de itens, como uma lista, tupla, string, etc.

Exemplo 1: Iterando sobre uma lista

frutas = ["maçã", "banana", "cereja"]
for fruta in frutas:
    print(fruta)

Saída:

maçã
banana
cereja

Explicação: O laço for percorre cada item da lista frutas e imprime seu valor.

Exemplo 2: Usando range() para gerar números

for i in range(5):
    print(i)

Saída:

0
1
2
3
4

Explicação: range(5) gera uma sequência de números de 0 a 4. O laço imprime cada número.

Quando usar o for:

  • Quando você sabe quantas vezes precisa repetir o código (por exemplo, percorrer uma lista de tamanho conhecido).
  • Para iterar sobre elementos de uma coleção ou sequência.

2. Laço while

O laço while executa um bloco de código repetidamente enquanto uma condição específica for verdadeira. Ele é útil quando o número de iterações não é conhecido antecipadamente e depende de uma condição dinâmica.

Sintaxe:

while condição:
    # Bloco de código a ser executado enquanto a condição for True
  • condição: Uma expressão booleana que é avaliada antes de cada iteração. O laço continua enquanto ela for True.

Exemplo 1: Contagem regressiva

contador = 5
while contador > 0:
    print(contador)
    contador -= 1

Saída:

5
4
3
2
1

Explicação: O laço continua executando enquanto contador > 0. A cada iteração, o valor de contador é decrementado em 1.

Exemplo 2: Solicitar entrada até uma condição ser atendida

resposta = ""
while resposta != "sim":
    resposta = input("Você quer continuar? (sim/não): ")
print("Obrigado por confirmar!")

Explicação: O laço continua pedindo ao usuário para digitar "sim" ou "não" até que ele digite "sim". Quando isso acontece, o laço termina e a mensagem final é exibida.

Quando usar o while:

  • Quando o número de iterações não é conhecido antecipadamente.
  • Para criar loops que dependem de uma condição que pode mudar durante a execução (por exemplo, entrada do usuário ou cálculos dinâmicos).

3. Controle de Fluxo: break e continue

Dentro de laços for e while, você pode usar os comandos break e continue para controlar o fluxo de execução.

  • break: Interrompe o laço imediatamente, saindo dele.
  • continue: Pula o restante do código dentro do laço para a iteração atual e vai para a próxima iteração.

Exemplo com break:

for i in range(10):
    if i == 5:
        break
    print(i)

Saída:

0
1
2
3
4

Explicação: Quando i é igual a 5, o break interrompe o laço, mesmo que ainda haja iterações restantes.

Exemplo com continue:

for i in range(5):
    if i == 2:
        continue
    print(i)

Saída:

0
1
3
4

Explicação: Quando i é igual a 2, o continue pula a impressão e vai para a próxima iteração.

4. Diferenças entre for e while

  • for:
    • Ideal para iterar sobre sequências conhecidas (listas, tuplas, strings, etc.).
    • O número de iterações é geralmente conhecido ou pode ser determinado antecipadamente.
    • Mais seguro contra loops infinitos, pois a sequência tem um fim definido.
  • while:
    • Usado quando o número de iterações depende de uma condição que pode mudar durante a execução.
    • Pode levar a loops infinitos se a condição nunca se tornar falsa, por isso exige cuidado.

Dica: Se você souber quantas vezes precisa repetir o código, use for. Se a repetição depender de uma condição dinâmica, use while.

5. Exemplos Práticos

Exemplo 1: Somar números de uma lista com for

numeros = [1, 2, 3, 4, 5]
soma = 0
for num in numeros:
    soma += num
print("A soma é:", soma)

Saída: A soma é: 15

Exemplo 2: Criar um padrão de texto com while

contador = 1
while contador <= 5:
    print("*" * contador)
    contador += 1

Saída:

*
**
***
****
*****

Exemplo 3: Usar break para sair de um loop infinito

while True:
    resposta = input("Digite 'sair' para encerrar: ")
    if resposta == "sair":
        break

Explicação: O laço while True cria um loop infinito, mas o break permite sair dele quando o usuário digita "sair".

6. Boas Práticas

  • Evite loops infinitos: Sempre garanta que a condição do while eventualmente se torne falsa. Use break com cuidado em loops infinitos.
  • Mantenha o código simples: Evite aninhar muitos laços, pois isso pode tornar o código difícil de ler e manter.
  • Use for quando possível: É mais seguro e legível para iterações sobre sequências conhecidas.
  • Inicialize variáveis corretamente: No while, certifique-se de que as variáveis usadas na condição sejam inicializadas antes do laço.
  • Teste as condições: Verifique se as condições estão corretas para evitar comportamentos inesperados.

Conclusão

Os laços de repetição são ferramentas poderosas em Python, permitindo que você automatize tarefas repetitivas de forma eficiente. O for é ideal para iterar sobre sequências conhecidas, enquanto o while é mais flexível para situações onde o número de iterações depende de uma condição. Comandos como break e continue oferecem controle adicional sobre o fluxo dos laços.

Para dominar os laços de repetição, pratique com exemplos simples e progressivos, como os apresentados aqui. Tente criar seus próprios loops para resolver problemas do dia a dia, como processar listas de dados ou gerar padrões. Com o tempo, você se sentirá confortável usando for e while em diversas situações de programação!
