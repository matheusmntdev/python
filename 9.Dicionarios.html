Dicionários no Python

Os dicionários em Python são uma estrutura de dados que permite armazenar coleções de dados na forma de pares chave-valor. Eles são mutáveis, não ordenados (até Python 3.6; a partir do 3.7, mantêm a ordem de inserção) e extremamente úteis para mapear informações de forma eficiente. Diferentemente de listas ou tuplas, que usam índices numéricos para acessar elementos, os dicionários usam chaves únicas, que podem ser de qualquer tipo imutável (como strings, números ou tuplas).

Vamos explorar como criar, manipular e usar dicionários, suas principais características e exemplos práticos para consolidar o aprendizado.

1. O que é um Dicionário?

Um dicionário é uma coleção de pares chave-valor, onde:

  • Chave: Um identificador único (imutável, como strings, números ou tuplas).
  • Valor: O dado associado à chave (pode ser qualquer tipo de dado: número, string, lista, outro dicionário, etc.).

Características dos Dicionários

  • Mutáveis: Você pode adicionar, remover ou modificar pares chave-valor.
  • Não ordenados (até Python 3.6): Não possuem índice fixo como listas; a partir do Python 3.7, mantêm a ordem de inserção.
  • Chaves únicas: Não pode haver chaves duplicadas; se uma chave for repetida, o último valor sobrescreve o anterior.
  • Sintaxe: Definidos com chaves {}.

2. Criação de Dicionários

Existem várias formas de criar dicionários em Python:

Método 1: Usando chaves {} diretamente

# Dicionário vazio
dicionario_vazio = {}

# Dicionário com valores
pessoa = {"nome": "Ana", "idade": 25, "cidade": "São Paulo"}

Método 2: Usando a função dict()

pessoa = dict(nome="Ana", idade=25, cidade="São Paulo")

Método 3: Com uma lista de tuplas

pessoa = dict([("nome", "Ana"), ("idade", 25), ("cidade", "São Paulo")])

3. Operações Comuns com Dicionários

Dicionários oferecem várias operações para manipular dados. Aqui estão as mais comuns:

1. Acessar valores

Use a chave para acessar o valor correspondente.

pessoa = {"nome": "Ana", "idade": 25}
print(pessoa["nome"])  # Saída: Ana
print(pessoa["idade"])  # Saída: 25

Nota: Se a chave não existir, um erro KeyError será gerado. Para evitar isso, use o método get():

print(pessoa.get("cidade", "Não informada"))  # Saída: Não informada (valor padrão)

2. Adicionar ou modificar valores

Atribua um valor a uma chave. Se a chave já existe, o valor será atualizado; se não, a chave-valor será adicionada.

pessoa["cidade"] = "São Paulo"  # Adiciona nova chave-valor
pessoa["idade"] = 26  # Atualiza o valor existente
print(pessoa)  # Saída: {'nome': 'Ana', 'idade': 26, 'cidade': 'São Paulo'}

3. Remover elementos

  • pop(): Remove uma chave e retorna o valor associado.
  • popitem(): Remove e retorna o último par chave-valor.
  • del: Remove uma chave específica.
# Usando pop()
idade = pessoa.pop("idade")
print(idade)  # Saída: 26
print(pessoa)  # Saída: {'nome': 'Ana', 'cidade': 'São Paulo'}

# Usando popitem()
ultimo = pessoa.popitem()
print(ultimo)  # Saída: ('cidade', 'São Paulo')
print(pessoa)  # Saída: {'nome': 'Ana'}

# Usando del
pessoa["profissao"] = "Engenheira"
del pessoa["profissao"]
print(pessoa)  # Saída: {'nome': 'Ana'}

4. Verificar se uma chave existe

Use o operador in para verificar a existência de uma chave.

if "nome" in pessoa:
    print("A chave 'nome' existe!")
# Saída: A chave 'nome' existe!

5. Obter todas as chaves, valores ou pares

  • keys(): Retorna uma visão das chaves.
  • values(): Retorna uma visão dos valores.
  • items(): Retorna uma visão dos pares chave-valor.
pessoa = {"nome": "Ana", "idade": 25, "cidade": "São Paulo"}
print(pessoa.keys())   # Saída: dict_keys(['nome', 'idade', 'cidade'])
print(pessoa.values()) # Saída: dict_values(['Ana', 25, 'São Paulo'])
print(pessoa.items())  # Saída: dict_items([('nome', 'Ana'), ('idade', 25), ('cidade', 'São Paulo')])

6. Tamanho do dicionário

Use len() para contar o número de pares chave-valor.

print(len(pessoa))  # Saída: 3

4. Iterando sobre Dicionários

Você pode iterar sobre dicionários de várias formas:

1. Iterar sobre chaves (padrão)

for chave in pessoa:
    print(chave)

Saída:

nome
idade
cidade

2. Iterar sobre valores

for valor in pessoa.values():
    print(valor)

Saída:

Ana
25
São Paulo

3. Iterar sobre pares chave-valor

for chave, valor in pessoa.items():
    print(f"{chave}: {valor}")

Saída:

nome: Ana
idade: 25
cidade: São Paulo

5. Dicionários Aninhados

Dicionários podem conter outros dicionários como valores, criando estruturas aninhadas.

Exemplo: Cadastro de usuários

usuarios = {
    "user1": {"nome": "Ana", "idade": 25, "email": "ana@email.com"},
    "user2": {"nome": "Bruno", "idade": 30, "email": "bruno@email.com"}
}
print(usuarios["user1"]["nome"])  # Saída: Ana

Adicionando novos dados

usuarios["user3"] = {"nome": "Clara", "idade": 22, "email": "clara@email.com"}
print(usuarios)

6. Exemplos Práticos

Exemplo 1: Contagem de frequência

Crie um programa que conte a frequência de palavras em uma frase.

frase = "banana maçã banana maçã laranja"
palavras = frase.split()
frequencia = {}

for palavra in palavras:
    if palavra in frequencia:
        frequencia[palavra] += 1
    else:
        frequencia[palavra] = 1

print(frequencia)  # Saída: {'banana': 2, 'maçã': 2, 'laranja': 1}

Exemplo 2: Cadastro de notas de alunos

Gerencie notas de alunos em um dicionário.

notas = {"Ana": 8.5, "Bruno": 7.0, "Clara": 9.0}
notas["Ana"] = 9.0  # Atualiza nota
notas["Daniel"] = 6.5  # Adiciona novo aluno

for aluno, nota in notas.items():
    print(f"{aluno}: {nota}")

Saída:

Ana: 9.0
Bruno: 7.0
Clara: 9.0
Daniel: 6.5

Exemplo 3: Dicionário aninhado para dados estruturados

Armazene informações de produtos em uma loja.

produtos = {
    "001": {"nome": "Notebook", "preco": 3500, "estoque": 10},
    "002": {"nome": "Mouse", "preco": 50, "estoque": 100}
}
print(produtos["001"]["nome"])  # Saída: Notebook
produtos["001"]["estoque"] -= 1
print(produtos["001"])  # Saída: {'nome': 'Notebook', 'preco': 3500, 'estoque': 9}

7. Métodos Úteis de Dicionários

Aqui estão alguns métodos importantes para manipular dicionários:

  • update(): Atualiza o dicionário com pares chave-valor de outro dicionário ou iterável.
  • clear(): Remove todos os itens do dicionário.
  • copy(): Cria uma cópia rasa do dicionário.
# Exemplo com update()
d1 = {"a": 1, "b": 2}
d2 = {"b": 3, "c": 4}
d1.update(d2)
print(d1)  # Saída: {'a': 1, 'b': 3, 'c': 4}

# Exemplo com clear()
d1.clear()
print(d1)  # Saída: {}

# Exemplo com copy()
original = {"x": 10}
copia = original.copy()
copia["x"] = 20
print(original)  # Saída: {'x': 10}
print(copia)     # Saída: {'x': 20}

8. Boas Práticas

  1. Use chaves descritivas: Escolha nomes de chaves que reflitam o significado dos dados (ex.: "nome" em vez de "n").
  2. Evite chaves mutáveis: Chaves devem ser imutáveis (strings, números, tuplas). Evite listas ou dicionários como chaves.
  3. Use get() para evitar erros: Prefira dicionario.get(chave, valor_padrao) para evitar KeyError.
  4. Mantenha o código legível: Para dicionários grandes ou aninhados, formate o código para facilitar a leitura.
  5. Evite modificar dicionários durante iteração: Isso pode causar erros; se necessário, itere sobre uma cópia ou crie um novo dicionário.

9. Conclusão

Os dicionários são uma das estruturas de dados mais versáteis em Python, permitindo mapear dados de forma eficiente com pares chave-valor. Eles são ideais para representar dados estruturados, como cadastros, configurações ou contagens, e oferecem métodos poderosos para manipulação e consulta.

Pratique criando dicionários para resolver problemas reais, como organizar informações ou contar frequências, para solidificar seu entendimento. Com o tempo, você desenvolverá um senso natural de quando e como usar dicionários em seus projetos!
