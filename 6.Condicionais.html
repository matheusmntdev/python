Condicionais no Python

As condicionais em Python são estruturas de controle que permitem executar diferentes blocos de código dependendo de condições específicas. Elas são fundamentais para criar lógica de decisão em programas, possibilitando que o fluxo de execução mude com base em valores de variáveis ou entradas do usuário. Em Python, as condicionais são implementadas principalmente com as palavras-chave if, elif e else.

Estrutura Básica do if

O if verifica uma condição e, se ela for verdadeira, executa o bloco de código associado.

idade = 18
if idade >= 18:
    print("Você é maior de idade.")

Se a condição for falsa, o código dentro do if é ignorado.

Uso do else

O else é executado quando a condição do if é falsa.

idade = 16
if idade >= 18:
    print("Você é maior de idade.")
else:
    print("Você é menor de idade.")

Uso do elif

O elif permite testar múltiplas condições em sequência.

nota = 85
if nota >= 90:
    print("Aprovado com distinção.")
elif nota >= 60:
    print("Aprovado.")
else:
    print("Reprovado.")

Exemplo Prático

Veja um exemplo que verifica se um número é par ou ímpar:

numero = 7
if numero % 2 == 0:
    print("O número é par.")
else:
    print("O número é ímpar.")

Conclusão

As condicionais são essenciais para controlar o fluxo de um programa em Python. Com elas, você pode criar lógica para tomar decisões baseadas em condições específicas, usando if, elif e else.
