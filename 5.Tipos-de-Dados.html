Tipos de Dados no Python

Os tipos de dados em Python definem o tipo de valor que uma variável pode armazenar e as operações que podem ser realizadas com esses valores. Python é uma linguagem de tipagem dinâmica, o que significa que o interpretador infere o tipo de uma variável com base no valor atribuído a ela, sem a necessidade de declarar explicitamente o tipo. Isso oferece flexibilidade, mas também exige cuidado para evitar erros de tipo em tempo de execução.

Neste conteúdo, vamos explorar os principais tipos de dados em Python, suas características e como utilizá-los.

Principais Tipos de Dados em Python

  • Números: Inteiros (int), Ponto flutuante (float), Complexos (complex)
  • Sequências: Strings (str), Listas (list), Tuplas (tuple)
  • Conjuntos: Conjuntos (set), Conjuntos imutáveis (frozenset)
  • Mapeamentos: Dicionários (dict)
  • Booleanos: Booleanos (bool)
  • Nenhum: NoneType (None)

1. Números

Os tipos numéricos em Python são usados para representar valores numéricos, como inteiros, decimais e números complexos.

  • Inteiros (int): Representam números inteiros, positivos ou negativos, sem parte decimal. Em Python, não há limite prático para o tamanho de um inteiro, exceto pela memória disponível.
    • Exemplo: x = 10
    • Uso: Contagens, índices, operações matemáticas inteiras.
  • Ponto flutuante (float): Representam números com parte decimal, usando o ponto como separador.
    • Exemplo: y = 3.14
    • Uso: Cálculos que exigem precisão decimal, como medidas e valores monetários.
  • Complexos (complex): Representam números complexos, com uma parte real e uma parte imaginária (indicada por j).
    • Exemplo: z = 1 + 2j
    • Uso: Cálculos avançados em engenharia e física.

2. Sequências

Sequências são coleções ordenadas de itens, que podem ser de diferentes tipos. Elas suportam indexação e fatiamento.

  • Strings (str): Sequências imutáveis de caracteres Unicode. São usadas para armazenar texto.
    • Exemplo: s = "Olá, mundo!"
    • Características: Imutáveis (não podem ser alteradas após a criação), suportam concatenação e fatiamento.
    • Uso: Manipulação de texto, entrada e saída de dados.
  • Listas (list): Coleções ordenadas e mutáveis de itens, que podem ser de diferentes tipos.
    • Exemplo: lista = [1, "dois", 3.0]
    • Características: Mutáveis (podem ser alteradas), suportam adição, remoção e modificação de elementos.
    • Uso: Armazenar coleções de dados que precisam ser modificadas, como listas de itens.
  • Tuplas (tuple): Coleções ordenadas e imutáveis de itens, semelhantes às listas, mas não podem ser alteradas após a criação.
    • Exemplo: tupla = (1, "dois", 3.0)
    • Características: Imutáveis, mais eficientes em termos de memória do que listas.
    • Uso: Armazenar dados que não devem ser alterados, como coordenadas ou registros fixos.

3. Conjuntos

Conjuntos são coleções não ordenadas de itens únicos, úteis para operações de conjunto como união e interseção.

  • Conjuntos (set): Coleções mutáveis de itens únicos, sem ordem definida.
    • Exemplo: conjunto = {1, 2, 3}
    • Características: Não permitem duplicatas, suportam operações de conjunto.
    • Uso: Eliminar duplicatas, verificar pertinência, realizar operações de conjunto.
  • Conjuntos imutáveis (frozenset): Versão imutável dos conjuntos, não podem ser alterados após a criação.
    • Exemplo: frozenset([1, 2, 3])
    • Uso: Quando é necessário um conjunto que não pode ser modificado, como chaves de dicionário.

4. Mapeamentos

Mapeamentos são coleções de pares chave-valor, onde cada chave é única e mapeada para um valor específico.

  • Dicionários (dict): Coleções mutáveis de pares chave-valor, onde as chaves devem ser imutáveis (como strings ou tuplas).
    • Exemplo: dicionario = {"nome": "João", "idade": 25}
    • Características: Acesso rápido aos valores via chaves, mutáveis.
    • Uso: Armazenar dados estruturados, como informações de usuários ou configurações.

5. Booleanos

Os booleanos representam valores de verdade e são usados em expressões condicionais e lógicas.

  • Booleanos (bool): Podem ser True ou False.
    • Exemplo: verdadeiro = True
    • Uso: Controlar o fluxo do programa, como em estruturas if, while, etc.

6. Nenhum

O tipo NoneType representa a ausência de valor ou um valor nulo.

  • None: Um valor especial que indica que uma variável não tem valor atribuído.
    • Exemplo: nada = None
    • Uso: Inicializar variáveis, indicar ausência de retorno em funções, etc.

Tipagem Dinâmica em Python

Em Python, os tipos de dados são dinâmicos, o que significa que você não precisa declarar o tipo de uma variável explicitamente. O interpretador infere o tipo com base no valor atribuído. Por exemplo:

x = 10       # x é um int
x = "dez"    # agora x é um str

Essa flexibilidade permite que você mude o tipo de uma variável durante a execução do programa, mas também requer atenção para evitar erros de tipo em tempo de execução.

Para verificar o tipo de uma variável, você pode usar a função type():

x = 10
print(type(x))  # <class 'int'>

Conversão de Tipos (Casting)

Python permite converter explicitamente entre diferentes tipos de dados usando funções como int(), float(), str(), etc. Isso é útil quando você precisa garantir que uma variável seja de um tipo específico.

Exemplo de conversão:

a = "10"       # string
b = int(a)     # converte para int: 10
c = float(a)   # converte para float: 10.0

Imutabilidade

Alguns tipos de dados em Python são imutáveis, o que significa que seu valor não pode ser alterado após a criação. Exemplos incluem strings, tuplas e frozensets. Para "alterar" um objeto imutável, você precisa criar um novo objeto.

Exemplo com strings:

s = "Olá"
s = s + ", mundo!"  # cria uma nova string

Tipos de Dados Avançados

Além dos tipos básicos, Python suporta tipos de dados mais avançados, como:

  • Classes personalizadas: Você pode definir seus próprios tipos de dados usando classes.
  • Tipos de coleções especiais: Como namedtuple, deque, etc., disponíveis no módulo collections.
  • Tipos numéricos adicionais: Como Decimal para precisão decimal exata e Fraction para frações.

Esses tipos avançados permitem maior flexibilidade e controle em cenários específicos.

Conclusão

Os tipos de dados em Python são fundamentais para armazenar e manipular informações de maneira eficiente e segura. Conhecer os diferentes tipos — como números, sequências, conjuntos, mapeamentos, booleanos e None — e suas características é essencial para escrever código claro e livre de erros. A tipagem dinâmica do Python oferece flexibilidade, mas também requer atenção para garantir que as operações sejam realizadas com os tipos corretos. Dominar os tipos de dados é um passo crucial para se tornar um programador Python proficiente!
